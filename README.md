# SiroMiku
為了自動化管理 Discord 伺服器而生<br>
可以用 反應身分組、訊息認證身分組、骰子<br>
我寫到 1000 行真的是不知道怎麼修<br>
一堆Bug，最好的建議就是給管理員權限。<br>

# 怎麼使用?
你需要 ``nodejs``<br>

```
npm i
```

```
node Command.js
```

在此之前，你還需要去獲取機器人 token 和 client ID，然後貼在config.json裡面。<br>
